Install

DXC Angular Components is a library made for Angular applications. In order to use it, you need to include it in an existing Angular project.

If you are creating a Angular application from scratch, we recommend using Angular setup to set it up with minimum configuration.

DXC Angular Components is distributed as an npm package. To use it in an existing project:

The CDK uses Angular Material so if it's not added, the components will not work as expected.

Furthermore, it is mandatory to add in your angular.json the library assets as well as the indigo pink pre-built theme from Angular Material:

Lastly, it is also needed our ThemeModule in your app so our default theme can be applied with the theme directive. For more information, please go to Custom Themes tab.
